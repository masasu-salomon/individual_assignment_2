# Kigali City Services & Places Directory

Flutter app for browsing and managing service/place listings in Kigali. Uses Firebase Auth and Firestore. State management with Provider.

## Run

```bash
flutter pub get
flutter run
```

Add your Firebase config (google-services.json, firebase_options) and enable Auth + Firestore in the Firebase project. On Android, add your app's SHA-1 in Firebase Console for sign-up to work.
